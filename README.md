# AI-Math

This repository is my personal Python learning journal. I created these scripts as I explored the basics of programming, working through input handling, math operations, conditionals, and data structures from my own point of view.

## My Story

I started this collection as a way to capture the small projects and experiments that helped me understand Python. Each file represents a moment in my learning process, from learning how to print a variable to building a small console tool that sanitizes input and generates a temporary password.

## What I Built

- `variables.py` — I practiced assigning values and printing them.
- `data_type.py` — I explored Python types like strings, integers, floats, tuples, lists, and dictionaries.
- `type_casting.py` — I learned how to convert user input between strings, integers, and floats.
- `input.py` — I experimented with reading input and splitting multiple values from one line.
- `operators.py` — I documented the operators I used: arithmetic, comparison, logical, bitwise, assignment, membership, and identity.
- `bitwise_ops.py` — I played with bitwise operations on numbers and NumPy arrays.
- `conditional_stmts.py` — I used control flow to evaluate different conditions and decide which message to show.
- `elif_stmt.py` — I written a decision flow to determine candidate status based on scores and experience.
- `else_if.py` — I wrote a simple example that calculates totals and prints them.
- `leap_yr.py` — I built a utility that checks whether a year is a leap year.
- `compound_interest.py` — I created a mini financial dashboard to calculate investment growth and doubling time.
- `list_complexity.py` — I learned about list operations like append, insert, pop, and slice.
- `slicing.py` — I explored string slicing with examples from a sample text.
- `text_sanitizer.py` — I wrote a program to clean user name and ID input, then generate a username and temporary password.
- `bug_catcher.py` — I made a fun example that counts insects and prints totals.
- `ps.py` — I built a bill-splitting calculator for restaurant tips.
- `Day_1.py` — I saved a simple script that marks the beginning of my journey with a status message.
